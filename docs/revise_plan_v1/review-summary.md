4. **배포 조기 검증** — Eng 지적. Day 1에 skeleton 배포 완료.

## 주요 의견 분기 & 결정

### 아키텍처: 모노리스 vs 분리
- **Eng**: Next.js 풀스택 모노리스 권장 (배포 단순화)
- **Competition**: FastAPI는 Python AI 생태계 깊이를 보여줌
- **결정**: FastAPI 유지 (AI 리포트에서 Python AI 활용 어필) + Supabase/Render로 배포 단순화

### TDD 유무
- **Eng**: 공모전에서 TDD는 시간 낭비
- **결정**: TDD 전량 제거 → 프롬프트 튜닝 + UI 폴리싱에 재투자

### 이미지 업로드 (Claude Vision)
- **Competition**: 멀티모달 AI 활용 = 점수 급상승
- **Eng**: 추가 공수 대비 효과 불확실
- **결정**: Nice-to-have (시간 여유 시 추가)

### Tool Use vs asyncio.gather
- **Eng**: asyncio.gather로 2회 병렬 호출 가능
- **Competition**: Tool Use가 AI 에이전트 아키텍처로 더 인상적
- **결정**: Tool Use 채택 (1회 호출 + 기술적 깊이 어필)
